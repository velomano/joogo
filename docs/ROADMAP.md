# Roadmap

## 단기 (1~2주)
- npm 전환 최종 마무리 (README/스크립트 동기화)
- 헬스체크 API 구현 (`/api/health`)
- CSV 업로드 → 스캔앱 정보 매칭 안정화
- GitHub Issues 생성 및 문서와 연동

## 중기 (1~2달)
- 멀티테넌트 로그인/권한 제어 완성
- 재고/피킹/출고 프로세스 고도화
- 관리자 KPI/과금 체계 도입
- Retool 관리자 페이지 확장 (필터/권한)

## 장기 (3개월+)
- 외부 OMS(셀메이트/이지어드민) API 연동 자동화
- SaaS 형태 배포 및 외부 고객 온보딩
- 모바일 스캔앱 오프라인 동기 강화
- 로그/분석 시스템 구축 (비용/성과 시각화)
